#  Netflix Landing Page Clone

Este é um projeto de estudo desenvolvido para praticar conceitos fundamentais de desenvolvimento Front-end, focado em **HTML5** e **CSS3**. O objetivo foi replicar a interface da página inicial da Netflix, priorizando a semântica e o layout responsivo com Flexbox.

---

##  Tecnologias Utilizadas

O projeto foi construído utilizando:
- **HTML5**: Estruturação semântica de dados.
- **CSS3**: Estilização, filtros de imagem (linear-gradient) e posicionamento.

##  O que foi praticado:

- [x] **Centralização Total**: Uso de `display: flex` e `align-items: center` para posicionar elementos no centro da tela.
- [x] **Manipulação de Background**: Ajuste de imagens de fundo com `background-size: cover` e filtros para melhorar o contraste do texto.
- [x] **Componentização Visual**: Criação de botões e inputs personalizados seguindo um Guia de Estilo (Netflix).
- [x] **Reset de CSS**: Aplicação de `box-sizing: border-box` e remoção de margens padrão para um layout consistente.
